# Telegram Currency Bot

Telegram-бот на Python с использованием aiogram.

## Возможности
- FSM (анкета пользователя)
- Хранение данных в SQLite (SQLAlchemy ORM)
- Интеграция с внешним API (курс валют)
- Логирование и обработка ошибок

## Стек (комбинация языков, библиотек, фреймворков и инструментов)
- Python 3.11+
- aiogram
- SQLAlchemy
- aiohttp

## Запуск
1. Создать `.env` файл
2. Указать `BOT_TOKEN`
3. Запустить:
```bash
python bot.py
